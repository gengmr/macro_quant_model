# 宏观量化投研终端 (Macro Quant Terminal) 系统说明文档

本系统是一款为金融机构与专业投资者设计的宏观资产量化投研工具。系统基于 **“双区间实证隔离（Train-Test Split）”** 与 **“阶梯式风险剥离（Risk-Off）”** 模型，旨在彻底消灭传统回测中的“未来函数”与“过度拟合”问题。通过全方位的数据扫描、时序因果测定与动态仓位模拟，帮助投资者寻找具有真实前瞻预测价值的宏观经济指标，并构建能够有效规避系统性戴维斯双杀的量化防线策略。

---

## 1. 核心架构与设计理念

### 1.1 无未来函数的双区间隔离 (Train-Test Isolation)

* **痛点**：传统回测往往利用全部历史数据计算单一阈值，导致回测曲线完美，但实盘立刻失效（因为实盘交易时无法知道未来的数据均值与方差）。
* **本系统方案**：系统强制将时间划分为两个完全隔离的区间：
* **拟合校验区 (Train Period)**：仅在此区间内进行历史数据挖掘，寻找因子与资产的最优前瞻时间差（$Lag$）、传导方向（正/负相关），并固化该区间的基准期望（$\mu$）与标准波动率（$\sigma$）。
* **滚动验证测试期 (Test Period)**：完全模拟真实的实盘环境。在 $t$ 时刻，系统仅使用已经客观公布的（即落后于当前时点 $Lag$ 个月的）宏观数据，带入 Train 区间的 $\mu$ 和 $\sigma$ 进行打分决策。



### 1.2 动态阶梯仓位模型 (Level & Momentum Strategy)

* **痛点**：简单的“突破买入、跌破卖出”容易在震荡市中产生剧烈的摩擦损耗（Whipsaw）。
* **本系统方案**：引入了 **水位（Z-Score）** 与 **中线趋势动量（Momentum, $MA\_Z_t$）** 双重判定机制。通过“逢险降仓”的防守逻辑，系统并非只做 0% 或 100% 的极管切换，而是设置了 0%、30%、50%、80%、100% 的五级阶梯防线，在趋势放缓时锁定利润，在深度冰点时彻底离场。

## 2. 全局交互组件介绍

### 2.1 顶部全局控制台 (Global Control Bar)

这是驱动整个终端运算的核心枢纽，所有的参数变动都会实时、全局地同步刷新下方所有的面板数据与图表。

* **全局时间区间 (Global Time Horizon)**：
* **起止时间选择器**：设定整个系统（包括 Dashboard、矩阵、滚动相关性等）分析的数据边界。
* **“取公共时间”按钮**：智能功能。点击后，系统会扫描左侧侧边栏已勾选的所有指标（包含宏观和资产），自动计算出它们拥有数据的**最大公共交集时间**，并自动填入起止时间框内。


* **全局宏观时差 (Global Lag Slider)**：
* 范围从 **-12个月 到 +12个月**，默认为 0。
* 用于手动对宏观数据进行时间轴平移。调整后，仪表盘、相关性矩阵等模块的数据会对齐平移后的时间进行重新计算。正值表示宏观领先，负值表示宏观滞后。*(注：在“双区间量化回测”模块中，此参数会被系统根据算力寻优自动接管，不再生效。)*


* **价格口径 (Price Mechanism)**：
* 支持切换资产定价锚点：**月末收盘价**、**月均价**、**月初开盘价**。改变后所有涨跌幅与估值逻辑均会基于新口径重构。


* **主题与视觉辅助**：
* **A- / A+**：全局调整界面的基础字号（Base Rem），适配不同分辨率与视距。
* **🌗 主题切换**：一键无缝切换 “极客暗黑 (Dark)” 与 “商务亮白 (Light)” 两套专业级色彩主题。



### 2.2 侧边栏指标库 (Data Catalog Sidebar)

支持通过顶部 `☰` 按钮一键**横向折叠/展开**，为核心图表腾出最大视觉空间。

* **智能搜索框**：支持全拼、缩写、代码模糊搜索。
* **宏观经济数据库**：蓝底标签，涵盖 M1、M2、PMI、社融、PPI 等核心宏观序列。
* **资产指数库**：金底标签，涵盖沪深300、科创50、创业板指等宽基或行业指数。
* **数据时间戳标引**：每个指标下方以等宽字体标注了底层数据库中实际拥有的**起止生命周期**（如 `2000-01 至 2023-10`），有效防范选择“短命数据”导致回测失败。

## 3. 核心分析模块详解 (Tabs)

系统由五个渐进式的投研面板构成，从基础数据观察一步步深入至复杂的交易模型构建。

### Tab 1: 综合仪表盘 (Dashboard)

用于直观呈现数据的历史运行轨迹与基础定性关系。

* **Z-Score 标准化走势对比**：
* 将不同量纲的数据（如万亿级别的 M2 与百分比的 PMI）统一映射至零均值、单位方差的坐标系。
* 悬停卡片（Tooltip）：悬停任意节点，系统不仅会显示该点的 Z 得分，还会展示其在**底层数据库中的真实数值**以及**实际发生的时间期**（在设置了全局 Lag 的情况下尤为重要）。


* **全区间相关性矩阵 (Pearson R Matrix)**：
* 将已选指标两两交叉，输出全区间的 Pearson 线性相关系数。
* 自带专业评级标签（强相关、中度相关、弱相关、无显著相关），并用颜色冷暖映射其关联强度。



### Tab 2: 滚动相关分析 (Rolling Correlation)

检验因子的有效性是否具有“时变特征”（即在不同经济周期下，因子的解释力是否发生结构性崩溃）。

* **动态图谱呈现**：分别展示 6 个月、12 个月、24 个月的滚动窗口相关系数 $R_t$。若某因子在牛市正相关，在熊市转为极度负相关（曲线频繁穿越零轴），说明该因子不具备跨周期的稳定定价能力，需谨慎使用。

### Tab 3: 领先滞后分析 (Lead-Lag Analysis)

寻找能够指引右侧交易的“左侧发令枪”。

* **时空平移扫描**：系统在后台对宏观因子执行 $-12$ 到 $+12$ 个月的暴力平移，探测每个 $Lag$ 点的相关系数。
* **前瞻属性判定卡片**：
* **先行预测指标 ($Lag > 0$)**：宏观波动先于资产，是构建量化择时模型的圣杯。
* **同步共振指标 ($Lag = 0$)**：宏观与市场同频发力。
* **滞后确认指标 ($Lag < 0$)**：宏观公布晚于市场盘面反馈，这种因子**绝对不能**用于左侧预测交易，只能作为右侧趋势的追认。

### Tab 4: 双区间量化回测 (Signal Backtest Engine)

**（核心杀手锏级功能模块）**

该面板左侧为**参数配置区**（支持 `◀ 折叠左侧配置面板`），右侧为**深度复盘监控区**。

#### A. 参数配置区

1. **模型引擎类型**：
* **单因子模型**：只利用一个核心宏观指标穿透底层逻辑。
* **多因子等权雷达矩阵**：吸纳侧边栏勾选的所有宏观指标，在校验区分别扫描其前瞻 $Lag$ 并按预测置信度 $|R|$ 执行加权整合（公式见系统面板说明）。


2. **基准对标策略**：支持比较“100%满仓持有”和“永远50%半仓”两种被动基准。
3. **因子入池最低阈值 ($|R|_{min}$)**：剔除弱因子的滤网，范围 0.2~1.0。若校验区跑出的最大相关性低于此值，因子将被直接淘汰，强行阻断回测，防止“垃圾进，垃圾出”。
4. **时空切割比例 (Train vs Test)**：滑块调节。如 `60% : 40%`，意味着前 60% 的时间作为无交易的参数拟合期，后 40% 的时间用于真金白银的盲测验证。
5. **交易策略风险偏好**：
* **稳健型 (Robust)**：以控制最大回撤为最高纲领。入场门槛高，稍有风吹草动（动量放缓）便大幅砍仓至 50% 或 20%。
* **激进型 (Aggressive)**：博取高弹性 Alpha。对底部反转极其敏感，指标微弱复苏即敢于推高至 80% 甚至满仓。



#### B. 深度复盘监控区

1. **校验区因子特征溯源说明 (Train Period Features)**：
* **多维度定标表**：清晰展示系统在 Train 区间内，是如何一步步扫描不同的 $Lag$，并最终提取出最优的 $Lag$、固化的 $\mu$、$\sigma$ 及传导方向（正向同频/反向对冲）的。
* **预测强度雷达图**（多因子专有）：直观展现哪些因子是主力核心，哪些是平滑辅助（红色为正向，绿色为负向反转）。


2. **核心风控指标五联卡 (KPI Metrics)**：
* 实时汇总：策略累计净收益、超额 Alpha、年化夏普比率 (Sharpe)、历史最大亏本率 (Max DD，即跌破初始本金的极值比例)、调仓触发次数。


3. **验证期双轴资金曲线与防线规避图**：
* **左轴（曲线）**：策略的净值曲线与被动基准的净值曲线的厮杀。
* **右轴（浅蓝面积）**：系统在当前月分配给资产的实际持仓比例（0%~100% 阶梯）。
* **隐藏第三轴（单因子原值/多因子得分）**：可通过顶部 Checkbox 打开，观测因子的走势与调仓动作的直接映射。
* **灾难预警热力散点**：图表**仅在系统判断有风险并执行降仓（即持仓 < 100%）时标记散点**。散点的尺寸越大，说明减仓力度越狠；红色（买回/增仓），绿色（抛售/减仓）。散点上方带有 **✓ (成功避险或抄底)** 或 **✕ (踏空或被套)** 的正误评判印记。


4. **策略执行明细大表 (Trade Log)**：
* 按时间轴正序（由远及近）详细复盘每一个月的操盘细节。
* **决策逻辑推演**列是本系统的精髓，它会使用专业的投研语言，向你解释由于当前的 $Z_t$ 处于什么位置、$\Delta MA\_Z_t$ 呈现什么状态，从而导致了防线级别上调/下调，极其透明。



---

### Tab 5: 综合分析研报 (Comprehensive Report)

一键生成可直接交付或用于复盘总结的机构级宏观量化研报。

* **数据驱动，拒绝主观**：报告通过解析你在“双区间量化回测”中配置的超参数（Train-Test 时间节点、模型偏好、因子库），后台重新调度引擎执行极速推演。
* **四大章节结构化输出**：
1. 宏观环境概览与标的入库（交代数据清洗与 Z-Score 处理边界）。
2. 全局线性联动网络图谱（输出全量指标的相关性定级）。
3. 校验区时序因果与前瞻属性判定（明确哪些指标是真正的“左侧发令枪”）。
4. 量化模型回测与实战防线复盘（通过深度数据对比，撰写详实的**研报终审结论**，明确告知该因子组合是否具备跨越周期的战术配置价值，是否战胜了戴维斯双杀）。
