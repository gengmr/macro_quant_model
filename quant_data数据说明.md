# 量化宏观交叉分析终端 —— 数据规范与设计目标白皮书

## 一、 系统设计目标

本系统的核心业务目标是：**提供一个纯粹、高效的环境，用于深度剖析“宏观经济数据”与“股市/金融资产数据”之间的客观关联性与驱动逻辑。** 为达成该核心愿景，系统在产品设计上需严格遵循以下三大核心宏观要求：

### 1. 视觉设计与交互体验

* **美观专业**：界面和各类组件样式需保持专业、优雅与高级感，符合金融量化分析产品的严谨调性。
* **高交互性与易用性**：操作逻辑需直观顺畅，功能响应迅速，确保用户在进行数据选择、参数配置和图表分析时获得极佳的易用体验。

### 2. 多维度专业关联性分析

* **科学的评估指标与图表**：需根据计算数据客观、合理地设计多维度的评估指标（如相关性计算、错位时间差分析等）与可视化图表（如趋势叠加、数据拟合等），以全面、清晰地展现宏观经济与股市之间的内在联系。
* **客观规范的结论生成**：系统需设定科学的判定规则，根据底层数据的计算结果，自动输出详细、清晰、规范的文字版相关性分析结论。

### 3. 合理的页面布局与空间利用

* **动态菜单布局**：合理规划整体页面空间，菜单栏/控制面板需支持“悬停展开、移开折叠”的动态交互。展开后的菜单必须支持完整滑动，确保所有宏观指标和股票类别的名称完整显示，彻底杜绝文字截断。
* **图表全屏与缩放**：为方便对长周期趋势和关联细节进行深度观测，所有图表分析区域必须支持独立放大至全屏显示的功能。

---

## 二、 底层数据结构规范

为了让系统能够极速、无缝地完成宏观与资产的多维度对齐与计算，输入系统的数据底座必须严格遵循以下两套 JSON 数据字典规范。

### 数据源 A：宏观经济数据 (Macro Indicators JSON)

**数据定位：** 作为分析的基准变量和绝对时间锚点。
**结构规范：** 采用 `大类板块 -> 具体指标名称 -> 年月时间(YYYY-MM) -> 数值` 的三层嵌套字典格式。

**格式示例：**

```json
{
  "宏观经济-货币线": {
    "居民消费价格指数同比(CPI)": {
      "2014-09": 1.6,
      "2014-10": 1.6,
      "2014-11": 1.4
    },
    "狭义货币供应量同比(M1)": {
      "2014-09": 4.8,
      "2014-10": 3.2
    }
  },
  "宏观经济-业绩线": {
    "采购经理指数(PMI)": {
      "2014-09": 51.1,
      "2014-10": 50.8
    }
  }
}

```

### 数据源 B：股票/金融资产月频数据 (Asset Monthly JSON)

**数据定位：** 作为分析的目标变量。
**结构规范：** 必须包含 `metadata` 声明层以供系统进行类型识别。资产数据按 `资产代码 -> 资产元数据 + 月度聚合数据(monthly_data) -> 年月时间(YYYY-MM) -> 多口径度量值` 进行四层嵌套。

**核心字段要求：**
为适配不同维度的宏观经济指标（流量或存量），每个月的股票资产数据必须提供以下三种重采样计算口径：

1. `close_eom` (月末收盘价)：用于对齐期末存量基准。
2. `close_avg` (月均收盘价)：用于对齐全月流量平滑基准。
3. `open_bom` (月初开盘价)：用于对齐前置观测基点。

**格式示例：**

```json
{
  "metadata": {
    "version": "1.0",
    "type": "monthly"
  },
  "data": {
    "159919.SZ": {
      "name": "沪深300ETF",
      "monthly_data": {
        "2014-09": {
          "open_bom": 2.45,
          "close_eom": 2.50,
          "close_avg": 2.48,
          "volume_sum": 154613.9,
          "amount_sum": 1546139.0
        },
        "2014-10": {
          "open_bom": 2.50,
          "close_eom": 2.55,
          "close_avg": 2.52,
          "volume_sum": 182345.1,
          "amount_sum": 1823451.2
        }
      }
    },
    "510050.SH": {
      "name": "上证50ETF",
      "monthly_data": {
        "...": "..."
      }
    }
  }
}
```
